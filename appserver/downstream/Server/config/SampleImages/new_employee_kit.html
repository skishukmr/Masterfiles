New Employee Kit
Description & Benefits
Selection of this kit will notify all relevant parties of the start date of the new employee including Human Resources (for employee orientation), Security (for badging), Data/Telecom (for network, phone and computer installation) and Facilities (for cubicle build out).


Includes :
  • HR Benefits and Orientation Kit
  • Dell Pentium Pro 200 w/ 64Mb RAM, 2.2Gb Drive
  • Lucent MLX-10D Telephone
  • Security Badging
  • Steelcase Modular Desk Set and Ergonomic Chair
