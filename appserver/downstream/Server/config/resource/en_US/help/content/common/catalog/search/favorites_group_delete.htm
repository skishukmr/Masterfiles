Tips   Delete a favorites group and items

Favorite groups are handy for organizing catalog items by projects, departments, or types (such as office supplies), and make it easy to find frequently ordered items, which saves you time creating requests in the future.

When you add an item to a favorites group, a star appears beside it in the catalog to make it easier to see.

Screen details

You delete favorites groups from either the Explore Catalog or the Add Items screen by selecting Favorites > Other.
 
  1. Select Options > Favorites.
  2. Select Other... from the Favorites pull-down menu.
  3. Check the check box for the group to delete.
  4. Click Delete Group to permanently delete the group and its items.
