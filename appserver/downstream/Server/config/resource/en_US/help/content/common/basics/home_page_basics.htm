Tips   About the Home page

You can access the Home page from any screen by clicking Home in the Command bar.

Links in the Navigation Panel flash at a set interval to alert you when you have pending tasks. For example, if you receive requests that require your approval, the Approve Requests link flashes for several seconds, and will continue to flash at intervals.

Screen details

To display the Home page screen, log in and, if required, select a partition in which to work.
 

The Home page provides quick navigation to key tasks and request status, using the links in the Navigation Panel and the central "swoosh" or image, and the buttons in the Command bar.

To display a dialog box with information about your current session, click the welcome or user link in the Command bar. Information, such as the time of your last visit, and the current partition (if applicable) or node, is displayed. The dialog box closes when you click anywhere on the Home page or the screen refreshes.

From the Home page you can create and work with requests and documents, and monitor their status; explore catalogs and folders; and run searches and reports. You can also read system news and other information posted on the Home page by your My Supply Cabinet system administrator.

If you logged in as an My Supply Cabinet system administrator, click Administrator in the Explore section of the Navigation Panel to go to My Supply Cabinet Administrator. You can edit objects only in the partition you specified when you logged in; you have read-only access to objects in other partitions. You can also access integration events, scheduled tasks, and rules in all partitions. To learn more about using My Supply Cabinet Administrator, see the online help in My Supply Cabinet Administrator.
