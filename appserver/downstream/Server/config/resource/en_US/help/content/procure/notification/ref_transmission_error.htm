Tips   Notification 20: Immediate Order Transmission Failure

If you receive this message, you must manually transmit the order, usually by phone, and then update the order status in My Supply Cabinet using the Order command.

For more information on order transmission problems, see The My Supply Cabinet Procurement Guide.

Related topics

About notification messages

Notification messages
 

When there are errors in transmitting an order, such as an email or fax failure, My Supply Cabinet emails a notification message to users who have the OrderFailureImmediateEmail permission (typically purchasing agents) to ensure the failure is noted and handled. The message indicates why the transmission failed.
