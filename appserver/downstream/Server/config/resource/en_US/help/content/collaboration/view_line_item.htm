Tips   View line item details

When viewing a collaboration request or proposal, you can view the details of its line item(s).

Related topics

View collaboration request details

Edit a collaboration request

 
Screen details

When viewing the request or proposal details, click Detail in the row for a line item to display the View Line Item screen. If you are editing a request, click Included in the row for a line item, and then choose View Details from the pull-down menu.
 
  1. Review the settings of this line item.
  2. Add a comment to the line item as needed. Note that this comment does not appear on the request's message board and does not send a notification.
  3. When you have finished viewing the line item details, click OK.
