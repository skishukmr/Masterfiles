Tips   View search results

Two view-related fields are hidden until you browse or search for an item. After browsing or searching, you see the Show Results (per page) field, an additional View By field, and the page refreshes using the new settings.

The ... notation indicates that there are more categories or suppliers than the ones currently listed on the screen. Drill down or use the search refinement options to view additional categories or suppliers.

For faster searches:

  • View by summary instead of detail: less information displays, which can save time displaying large results sets.
  • Select a smaller number from the Show Results pull-down menu. This produces more pages of results, but they display more quickly. If there are too many results, try refining the search further.

Explore catalogs
 

When searching for catalog items, you can view the results by category, price, relevance, or supplier, and display summary or detail information. You can also change the number of items that display per page.

  1. From the Explore Catalog screen or the Add Items screen, browse or search first to display hidden fields, then select:
    • View By Category to view items grouped by category, which can be helpful if you know that the item is located in a specific category, such as “Office Supplies.”
    • View By Price to view items by price (lowest price first), which makes it easy to find items that fit your budget.
    • View By Relevance to view items by relevance (most popular items first). Relevance is determined based on the number of times an item is ordered or is selected as a favorite. Items ordered more often or flagged as favorites receive a higher relevance ranking.
    • View By Supplier to view the search results grouped by supplier, in supplier) alphabetical order.
      • When all items for the supplier display, you see the message “All items found in this group are shown below...”
      • If more items exist for the supplier than can be displayed, you see a message similar to: “View all 17 items found in this group...”.
      • Click the link to display all items in the group.
      • If the items are not associated with a particular supplier, the Supplier field displays [Unspecified].
  2. Select an additional option from the second View By field:
    • View By Details to view line item detail information, including the item description, supplier name and part number, estimated time of arrival, and price.
    • View By Summary to view line item summary information, including a short description of the item and its price.
    • View By Details (multi) to view detail information and multi-select check boxes, to easily select and add multiple items.
      • Check the desired items.
      • Click Add to Cart to begin creating a request that contains the item(s).
      • Click Add to Favorites to add individual items to a favorites group.
    • View By Summary (multi) to view summary information and multi-select check boxes, to easily select and add multiple items.
      • Check the desired items.
      • Click Add to Cart to begin creating a request that contains the item(s).
      • Click Add to Favorites to add individual items to a favorites group.
  3. Select the Show Results (per page) number (by 10, 20, 50, or 100 items per page).
  4. When your search returns multiple pages of results, the page selector displays. Select a page number from the pull-down menu or click the right and left arrows to select the next or previous page.
