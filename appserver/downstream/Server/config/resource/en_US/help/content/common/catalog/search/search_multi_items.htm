Tips   Add multiple catalog items to a request

Use the multi-select check boxes to add multiple items at the same time.

Exception: Certain catalog items cannot be added to a requisition using the multi-select option. If you are ordering items that have business forms attached or items that contain fields that must be specified before ordering, you must add each item to your requisition individually. If you attempt to use multi-select to add more than one item that uses a business form, you see a warning saying that you cannot multi-select these catalog items.

Once you add items to the request, you can edit, copy, or delete them from the shopping cart.

When exploring the catalog to add items to a request, use the View By multi-select options to add multiple items at a time.

  1. Browse or search in the catalog for an item to add to the request. Once you open the catalog, the View By Details pull-down menu displays.
  2. Click the View By Details pull-down menu and select Summary (multi) or Details (multi) to display the multi-select check boxes and summary or detailed item information.
  3. Click the check boxes associated with items to add to the request.
  4. Click Add to Cart to add the items.
    • If you are exploring the catalog, you are asked to start creating a request.
    • Once you create the request, use the multi-select check boxes to select and add the items.
    • Once you add items, they display in your shopping cart on the View Line Items screen.
