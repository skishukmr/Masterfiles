Tips   Review details

My Supply Cabinet contains screens that are used to display the details of a specific value, such as a supplier, digital certificate, or commodity code.

Advanced tax fields display on screens where you view and reconcile invoices, and view invoice field details.

Screen details

To display the Review Details screen, click any value link, such as commodity code, on a screen.

 
  1. Review details about the value.
    • The ID and the name are the unique identifiers for a value, such as a commodity code, in the My Supply Cabinet database. Often, ID and Name are the same. The Description field can provide additional information about the value to help you better identify it.
    • Click any link to display more information.
  2. Click Done to return to the previous screen.
