Tips   Add a favorite item to a group

Favorite groups are handy for organizing catalog items by projects, departments, or types (such as office supplies), and make it easy to find frequently ordered items, which saves you time creating requests in the future. (Flagging an item as a favorite creates a shortcut to the item.)

When you add an item to a favorites group, a star appears beside it in the catalog to make it easier to see.

Screen details

You add a favorites item to a group from either the Explore Catalog or the Add Items screen. To add a favorite item, search or browse for an item and click Add to Favorites next to the item and choose Other.

  To add an item to an existing favorites group:
  1. Browse or search to locate the item.
  2. Click Add to Favorites and select the desired favorites group.
  3. Click the Add to Favorites link to see the group that contains the item (indicated by the check mark).

To create a favorites group and then add a favorite item to it:

  1. Browse or search to locate the item.
  2. Click Add to Favorites > Other.
  3. Enter a group name in the New group name field (50 characters maximum).
  4. Click Create Group. The new group name displays.
  5. Click Add Item.
