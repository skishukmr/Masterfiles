System errors

Warnings and errors
 

If My Supply Cabinet detects errors or warnings when you submit a document for approval, it displays the Approvable Error Screen with an explanation of the error or warning.

If the message is an error, you must correct the error before you can successfully submit the request. If the message is a warning, you can submit the request for approval and take care of the warning at your earliest convenience.
