Tips   Explore catalogs

Different fields, buttons, and links appear in the Explore Catalogs and Add Items screens depending on what actions you have taken on previous screens.

The ... notation indicates that there are more categories or suppliers than the ones currently listed on the screen. Drill down or use the search refinement options to view additional categories or suppliers.

Screen details

To display the Explore Catalog screen, click Catalogs on the Home page. When you click Add Items to add items to a requisition or to a contract-related request, this screen displays with the title Add Items.

 

When exploring the catalog to add an item to a request, you can:

  • Use a simple text search.
  • Add refinement options for a more advanced search.
  • Search in a favorites group for items you previously added.
  • View search results by category, price, relevance, or supplier, with summary or detail information.
  • View catalog item details for product, supplier, and other information.
  • Add multiple items to a request at the same time.
  • Select the number of search results that display.
  • Browse a category in the catalog tree.
  • Punch out to a supplier's Internet Catalog.
  • Reset (clear) all search fields.

Once you locate the desired item, click Add to Cart:

  • On the Create Requisition screen, begin creating a requisition and add the item to your shopping cart.
  • If the item contains fields that must be specified before ordering, an intermediate screen is sometimes displayed. Fill in the blank fields and then click OK to begin creating a requisition.
  • Click Add to Favorites to add the item to a favorites group.
