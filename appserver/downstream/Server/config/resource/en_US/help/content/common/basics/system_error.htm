Tips   About My Supply Cabinet system errors

If you experience a problem, log in again and try the transaction again before contacting your Help Desk.

Extremely large attachments can cause a system error to occur. In this case, try zipping or revising (reducing) the attachment.

Screen details

The My Supply Cabinet System Error screen is displayed if My Supply Cabinet is unable to perform a requested task.
 

If My Supply Cabinet is unable to perform the task you request (for example, display a screen or run a report), it displays a system error. When this occurs, follow the suggestion to correct the problem. If a suggestion is not provided:

  • To display error details, click the expand / collapse arrow. Copy and paste the details into a Microsoft Work document to discuss with your Help Desk.
  • To display the Home page, click Home.
  • To display the Login screen and log in again, click Login.
