About search refinement options

When searching for catalog items to add to a request, you can use the following search fields to create more powerful searches. To use these fields, click Options to display the Options pull-down menu and select a refinement option.

Option

 Description
Contract field Select an item from a contract that you have permission to access. If you do not have contract permissions, this field does not appear.
ETA (Days) field

Find local catalog items whose estimated time of arrival falls within the number of business days entered. This is useful for time-critical orders because you can focus on items that meet your schedule requirements.

Enter the range of days within which the item must be delivered. For example: ETA (Days): From: “7” To: “14”

Note: Days are business days, not calendar days.

Exact Phrase field

Find local catalog items whose descriptions contain the text exactly as you've entered it. The search results include only those catalog items whose descriptions contain the phrase exactly as it is entered in this field.

Example: “LCD Projector” finds “MultiSync LCD Projector” but does not find “LCD Video Projector”.

When refining a search, you can combine an exact phrase with other search criteria, such as keywords, supplier part numbers, and so on. The search engine uses an "And" search for this field, combining the exact phrase with any other criteria you have entered in the other refinement fields.

Enter a minimum of three characters (and a maximum of 200 characters) before searching.
Exclude Words field

Exclude local catalog items from the search when their descriptions contain any of the specified words. Enter a minimum of three characters (and a maximum of 200 characters) before searching.

Example: To find LCD projectors that are not desktop models, enter Exclude Words “Desktop” and Keywords (And) “LCD Projector”. This finds “MultiSync LCD Projector” and excludes “Desktop Video LCD Projector”.

Use a space to separate words to exclude from the search.

Example: “adapter monitor splitter”
Favorites field

Select a favorites group to display the items you've previously added, or:

  • Create a new favorites group.
  • Search within an existing favorites group.
  • Select and delete a favorites group.

When you select a favorites group, the Favorites field locks, indicating that your search is restricted to the items in that group. Click the lock to unlock the filter.

The “Recent Choices” label indicates that the pull-down menu contains your most recently used search values, rather than a constrained set of search values.

Keywords (And) field

Find local catalog items whose descriptions contain all keywords entered. This is the default type of keyword search.

Example: Enter “black mechanical pencil” to find only those items that contain all three words (in any order), regardless of manufacturer, lead size, or other attributes. Enter three characters minimum before you search.

To search for any keyword you enter, add the Keyword (Or) field from the Options pull-down menu and enter keywords in that field before you search.

There is a 60-character limit for text searches.
Keywords (Or) field

Find local catalog items whose descriptions contain any keywords entered.

Example: Enter “black mechanical pencil” to find items containing the word "black”, items containing the word “mechanical”, and items containing the word “pencil”.

This search can potentially return many more results than a Keywords (And) search using the same words.

There is a 60-character limit for text searches.
Language field

Display catalog items in only one language.

Some catalogs are oriented for international users, and contain multiple line items (each in a different language) for the same catalog item. If your catalog contains multi-language items, select a language from the pull-down menu to receive results in that language only.

Manufacturer field

Find local catalog items whose manufacturer matches the selected value. Select a manufacturer from the pull-down menu to display the manufacturer's items. You can also use the manufacturer field to narrow a search for an item.

Example: If you are searching for a pen, enter the keyword “pen” and search, then select a manufacturer, such as “Bic” to refine the results to display only Bic pens.

Mfg. Part Number field

Find local catalog items whose manufacturing part number matches the value entered. When searching for part numbers, letters and numbers are significant; all other characters and white space are ignored.

Enter at least three alphanumeric characters for wildcard searching. For example, if you enter 9153, the search returns MON9153 6 or AD9153.

To find a specific part number, enter the part number in quotes. For example, if you enter "123-456", the search returns: 123-456, but not 123-45, 123-4, -456, or 3456.
Options For Type field When you choose to search within a parametric type (group) the parametric type field is locked and additional search fields are listed as search options for the type. Enter additional values in these fields to further refine the parametric search.
Price fields Find local catalog items in a specific price range. Enter the upper or lower limit, or enter both upper and lower limits to narrow the results set. Enter numeric values in each field. Example:
  • To find projectors that cost $5,000 or less, enter “5000” in the To field.
  • To find projectors that cost $5,000 or more, enter “5000” in the From field.
  • To find projectors that cost from $5,000 to $7,000, enter “5000” in the From field and “7000” in the To field.

    Note: The From field is the first of the two price fields. It is not labeled.

When you enter a number, such as 5000, the correct format is automatically applied, based on your default currency. For example, 5000 would display as $5,000.00, €5,000.00EUR, and so on.

You can also select a different currency using the currency chooser.
Supplier field

Find catalog items and internet catalogs for a particular supplier. Select a supplier to display the supplier's items. You can use the supplier field with other search option fields (such as manufacturer and keyword) to narrow a search.

Example:

  1. Select a supplier, such as “BoiseCascade” from the supplier pull-down menu to display all of the items they supply.
  2. Select a manufacturer, such as “3M” from the manufacturer pull-down menu to display their line of 3M products.
  3. Type the keywords “yellow post-it notes” to refine the results.

If an item is not associated with a particular supplier, the Supplier field displays [Unspecified]. You might be asked to specify a supplier later.
Supplier Part Number field

Find local catalog items whose supplier part numbers match the value entered. Example: Enter “BTM0123” to find item “BTM0123”.

You can also enter a stem to find items whose numbers start with the same stem.

Example: Enter “BTM” to find all supplier parts in a sequence, such as “BTM00057, BTM00060, BTM00106, BTM00107”.

Type Name field

Find local catalog items with parametric data of a particular type. When your catalog contains parametric data items, use this field to display additional fields that enable you to search for parametric type attribute values.

Example: Select “Pipe” from the Type Name pull-down menu to display items of that type, and additional search fields that are appropriate for the items, such as:

  • Thickness:
  • Diameter:
  • Thread Type:
  • Weight per Foot:
  • Size:
