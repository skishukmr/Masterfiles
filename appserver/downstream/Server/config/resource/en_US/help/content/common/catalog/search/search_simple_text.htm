Tips   Use a simple text search for catalog items

For efficient simple text searches:

  • Use one or two key words to describe an item.
  • Use specific words instead of general ones.
  • For automatic wildcard searching, enter characters for text searches or alphanumeric characters for part number searches. By default, you must enter a minimum of three consecutive characters, but your company can set a different minimum.
  • Use a stem search to find items grouped together by part number.
  • For text searches, enter a maximum of 200 characters.

If your keyword search results in too many items, use one or more options to refine the search.

View search results

 
  1. Enter one or two keyword(s) in the Keywords field to describe the item. Example: binder blue. By default, only items containing all the words entered are returned. To search for items containing any of the words entered, choose Keywords (or) in the search refinement options.
  2. Click Search to display all catalog items that contain the keyword(s). If the View By selection is set to display results by Category, items found in different categories are displayed in subsections. For example:
    • 18 items found in: Uncategorized Items More items...
    • 8 items found in: Industrial Equipment & Supplies > Production Equipment > Supplies More items...
  3. Click More items... to display all items in the category.

To search within a category, subcategory, or a favorites group:

  • Click the category or subcategory, enter the search values, and click Search.
  • Select the group name from the Favorites pull-down menu, enter the search values, and click Search.
