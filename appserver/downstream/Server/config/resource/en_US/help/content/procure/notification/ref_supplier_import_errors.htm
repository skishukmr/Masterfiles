Tips   Notification 15: Supplier Data Import Failure

After the import problem is resolved, it's recommended that the catalog manager revalidate the file and look for new errors. If the import problem remains, or a new error occurs, the import fails and the catalog manager is notified again.

This message is sometimes sent for information only when duplicate catalogs attempt to load at the same time. In this case, no action is required; My Supply Cabinet handles the problem automatically and moves on.

About notification messages

Notification messages
 

When a failure occurs while attempting to import a supplier catalog or a type definition file, My Supply Cabinet emails a notification message to users who have the correct permission, typically catalog managers or administrators. The message indicates which import failed, the status of the catalog or type definition file import, such as Validation Error, the number of errors that occurred, and the supplier of the catalog or type definition.

The catalog manager (or other administrator) should use the Catalogs screen in the Catalog Manager workspace in My Supply Cabinet Administrator to obtain details about catalog errors. Obtain details about the type definition errors on the Type Definitions screens in the Catalog Manager workspace.
