Tips   Edit Internet Catalog items

You can edit certain information for a supplier’s Internet Catalog items in My Supply Cabinet. These include accounting, ship to, and other information that resides in My Supply Cabinet.

If you want to edit information such as quantity, you must return (or re-punch out) to the supplier’s Website. To edit one or more Internet Catalog items:

  1. Display the request on the Summary screen or on the View Line Items screen.
  2. Click the title link for the line item you want to edit. My Supply Cabinet returns you to the supplier's Internet Catalog..
  3. Or, at the end of the Line Item Details section, click Edit this Punchout Catalog Item. My Supply Cabinet returns you to the supplier’s Internet Catalog and displays your order.
  4. In the supplier’s Internet Catalog, change the order quantity as needed, update your request, and then place your order. When you return to My Supply Cabinet, the line has been updated on your request.
