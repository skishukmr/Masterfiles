Tips   Notification 24: Receipt Failure

Users with the ReceiveError permission can query and edit all receipts in My Supply Cabinet. Typically you can edit the receipt and correct the problem.

Related topics

About notification messages

Notification messages
 

When an error occurs during a receipt push, My Supply Cabinet emails a notification message to users who have the ReceiveError permission (typically purchasing agents). The message identifies the receipt number and the date of the push attempt.
