Screen details

To delete a line item, select a Internet Catalog line item's check box on the View Line Items screen and click Delete.

 

When deleting Internet Catalog items from a requisition:

  • Click Delete to delete all of the items previously selected from a supplier.
  • Click the title link of the item to go to the supplier's Website where you can modify line items and add new ones.
  • Click Cancel to return to the previous screen without changing the request.
