About the Home page

 

You can withdraw a request that you have submitted for approval or that has been approved (if a delay purchase until date was specified on the request). While viewing the request details, click Withdraw to withdraw the request.

When you withdraw a request, My Supply Cabinet notes the action on the History tab and removes all existing approvals. It also removes any approvers you had added to the approval flow. If you submit the request again, the approval process starts over.

After you withdraw a request, you can:

  • Click Edit to edit the request and submit it again.
  • Click View to view the request details.
  • Click Home to return to the Home page.
