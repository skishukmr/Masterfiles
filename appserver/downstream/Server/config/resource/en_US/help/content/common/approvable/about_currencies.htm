Tips   About currencies

The currency chooser lists all available currencies from which you can select a currency.

When searching for catalog items, you can use the currency chooser to temporarily change the currency used while searching. For example, if your default currency is USD and you temporarily change the currency in the price field to EUR, My Supply Cabinet uses the EUR amount as the basis for the search, and displays catalog items where the USD price(s) are equivalent to the EUR price(s) you enter.

If your company has enabled multi-currency reporting, values in reports are displayed in your default currency and in the transaction currency. Whenever possible, values are calculated using historical conversion rates; otherwise, current conversion rates are used. If your company has also enabled euro-equivalent reporting, multiple columns of currency data are displayed.

Whenever you're dealing with money spent on behalf of your company, that money must be handled on your company's books. Your default currency is associated with your set of financial books, such as “US Dollar” or “Japanese Yen.” If necessary, you can change your currency temporarily in an My Supply Cabinet session, using the currency chooser.

Related topics

Change the default locale and currency
 

You specify your preferred or default currency in your user preferences. If you don't specify a default currency, My Supply Cabinet uses the default currency specified by your My Supply Cabinet system administrator.

You can override the default currency for specific transactions, such as requisition, and select another currency (known as a transaction currency). For example, if you work in the United States your default currency is US dollars (USD), but when creating a requisition for services to be performed in Australia, you would specify expenses for that service using Australian dollars (AUD). Depending on your assigned groups, roles, and permissions, you can also specify transaction currencies when approving requests.

Currencies are displayed using a prefix or suffix so you can see at a glance the currency that is in use. For example, you might see "$3.54 USD" to indicate US dollars. When a request involves multiple currencies, click Toggle Currency in the Command bar to display all monetary values in the default currency. Click Toggle Currency again to redisplay the values in their transaction currencies.

Note: When you toggle currencies, My Supply Cabinet uses current conversion rates to calculate values. These rates can vary from those that were in effect when a line item was last updated on the request. In this case, the sum of the items might not match the displayed total cost of the request, even after you click the Update Total button. Item values are recalculated using current conversion rates only when you make a change to the line item.
