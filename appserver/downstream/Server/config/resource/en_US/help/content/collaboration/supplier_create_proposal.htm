Tips   Create a collaboration proposal

To collaborate with a buying organization on a request, you create one or more proposals that provide the details of how you would fulfill the request. If there are multiple line items on the request, you can include and exclude items as needed so that only the items you want to provide are included on the proposal.

If the buying organization revises a request after you have submitted a proposal for it, you must revise your proposal to address the revised request. If the buyer creates an alternative request, you can submit a proposal for that request as well as the original. However, if the buyer does not allow multiple proposals, only your most recently submitted proposal is active.

Screen details

To display the screens for creating a collaboration proposal, click Actions when viewing the collaboration request, and then click Respond With Proposal.

 
  1. In the Review and Select Items screen, select the collaborative line item(s) you want to include in the proposal by clicking Included next to those items. To exclude an item that is currently set to be included, click Excluded next to it. If an item is set to be excluded, check with the buying organization before including it.
  2. To edit a line item, click its Included or Excluded button, and then choose Edit from the pull-down menu.
  3. If the item requires additional details to be included in the proposal, click the warning link and specify the required details.
  4. If the item requires a candidate, click Add, and then specify the candidate details. Note that if you need to edit the line item (see step 2), be sure to do so before adding any candidates.
  5. Click Next.
  6. In the Complete And Submit Response screen:
    • Verify that the correct items are included on the proposal. Note that candidate details do not display on this page.
    • Optionally, enter a comment in the message box. This comment will appear on the message board.
    • Optionally, click Add and specify a file to attach to this proposal.
  7. Click Submit.
