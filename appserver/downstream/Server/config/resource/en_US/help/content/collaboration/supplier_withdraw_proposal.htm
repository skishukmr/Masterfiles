Tips   Withdraw a collaboration proposal

You can withdraw a collaboration proposal you submitted, even if the buying organization has already accepted it, as long as collaboration has not ended. When collaboration ends, you can no longer withdraw the proposal. Instead, you must reject the purchase order if you no longer want to fulfill this request, or change the purchase order.

If you withdraw the proposal in error, you can resubmit it. If the proposal was already accepted before you withdrew it, the buyer must accept it again after you resubmit it.

You can only withdraw a proposal you have submitted. If you have not yet submitted a proposal and want to delete it, edit the proposal, click Cancel, and then select the option to delete the request (this proposal).

Screen details

To display the Withdrawing Proposal screen, click the Actions button when viewing the collaboration request details, and then choose Withdraw Proposal.
 
  1. In the Withdraw Reason list, select a reason for withdrawing this proposal.
  2. Optionally, enter a further explanation of why you are withdrawing the proposal.
  3. Click OK.
