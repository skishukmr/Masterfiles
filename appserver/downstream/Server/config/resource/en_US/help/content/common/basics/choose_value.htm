Tips   Choose one or more values

You can choose multiple values in some cases. For example, several suppliers can be associated with a sourcing request, so you can choose multiple supplier values.

The users listed are those in the current partition.

The users listed in the Refer To chooser for an Invoice Reconciliation (IR) document are based on the exception type, user permissions, and configuration. (Invoice functionality will be implemented in Release 2.)

Screen details

To display the Choose Value screen, select Other from a pull-down menu, or click [select] on a screen.
 
  1. Select a value from those initially displayed in the Add section or enter text and click Search to display all values that match the search text.
    • When available, you can specify a value for a specific field to search from the Field pull-down menu.
    • If your search yields no results, click Reset to redisplay all possible values. Then enter new text and click Search.
    • As an alternative to searching when multiple pages of values exist, click an arrow to display the previous or next page of values or select a page from the pull-down menu. When you navigate to another page, any values selected in the Add section are automatically added to the list of current values in the Current section.
    • If available, click the Table Options icon, located in the row above the column headers, to display commands that control the display of data in the table. For example, you can hide or show columns of data or group data.
    • Some rows provide icons for expanding () or collapsing ( ) rows of subcategory information.
  2. When you locate a value you want to use, select the value or click Cancel to return to the previous screen without selecting a value.
    • Where you can choose only one value, click Select to return to the previous screen and display the selected value.
    • Where you can choose multiple values, select the check box for each value and then click OK to return to the previous screen and display the selected values.
    • In both cases, the values you selected will appear in the list of current values in the Current section the next time you display the screen.
