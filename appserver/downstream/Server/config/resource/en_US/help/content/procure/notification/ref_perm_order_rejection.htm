Tips   Notification 64: Permanent Order Rejection from Ariba SN

If you are the requester, this is a courtesy message and does not require any action on your part.

Related topics

About notification messages

Notification messages
 

When an order fails and has been retried for the specified number of days, My Supply Cabinet emails this notification message to the requester of the items on the purchase order, and to users who have the permission OrderFailureImmediateEmail (typically purchasing agents).

To resolve the rejection, the purchasing agent (or another administrator) must edit the purchase order in My Supply Cabinet to fix the problem, and then run the task ProcessPendingOrdersWithURLSendMethod to move the order into the OrderFailed state, so it can be retried by the FailedOrders scheduled task.
