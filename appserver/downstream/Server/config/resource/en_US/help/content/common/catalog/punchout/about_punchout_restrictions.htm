Tips   About Internet Catalog restrictions

My Supply Cabinet displays a message if you try to return to a Internet Catalog that does not support re-punch out.

The actions you can take on Internet Catalog items previously added to a request depend on how the supplier has configured its Internet Catalog. For example, you must return (or re-punch out) to an Internet Catalog to edit the quantity of requisition items, if the supplier has enabled those actions, or you might be able to only view descriptions of the items.

My Supply Cabinet does not allow you to copy items from a supplier’s Internet Catalog you previously added to a request, because the price or other information may have changed since you initially added the items. When you want to add new Internet Catalog items or modify existing ones, you can return to the supplier’s site to update the information on your request. This ensures that your request contains the correct information.

If you copy a request that contains Internet Catalog items, My Supply Cabinet creates a copy of your original request and omits any Internet Catalog line items from the copy. If you try to copy an existing Internet Catalog line item to add to your existing request, My Supply Cabinet displays the message, “You cannot copy the selected items.”
