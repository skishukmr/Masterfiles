Tips   Resubmit a request

When you resubmit a denied request, the status reverts from Denied to Submitted.

Related topics

View request details

Requisition statuses
 

You can resubmit a request that has been denied by an approver. While viewing the request details, click Resubmit to enter comments that might clarify issues and result in approval. Click OK to resubmit the request or click Cancel to return to the previous screen without resubmitting the request.

When you resubmit a request, My Supply Cabinet notes the action on the History tab and routes the resubmitted request to the first approver in the approval flow. The resubmitted request must again go through all approvals.

After you resubmit a request, you can:

  • Click Print to print the request and exit. The request appears as a formatted page in your browser window. Select File > Print in the browser window to print the request.
  • Click same to create another request of the same type (for example, another requisition)
  • Click different to display the Create New Request screen and select a different type of request to create (for example, an expense report)
  • Click View to view the request details.
  • Click Home to return to the Home page.
