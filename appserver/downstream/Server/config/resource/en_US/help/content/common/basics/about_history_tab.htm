Tips   About the History tab

The History tab does not include actions taken while a request has a status of Composing. The History tab records actions only after the status has changed to Submitted.

You must have edit access to contracts to view the History tab when working with contracts.

Requisition statuses

History tab actions for contracts and requests
 

The History tab summarizes significant actions that have occurred to a document, including the date of each action and the user who took the action, if applicable. For example, depending on the document type, its status, and the permissions assigned to the user viewing the History tab, the tab might indicate:

  • Submittals, approvals (including escalations), denials, and withdrawals
  • Status changes
  • Accounting changes
  • Receipt and invoice reconciliation activities (Receiving and Invoice functionality will be implemented in Release 2.)
  • Comments or attachments added to the document
  • Problems or errors, such as order failures or oversize attachments
  • Conversion of currencies to the Euro

History tab items typically are sorted by date, with the most recent item at the top of the list.

To review an item:

  • Click the links to display details about users and requests.
  • Click a column heading to change the order by which items are sorted.
  • Click Print to print the status details.
