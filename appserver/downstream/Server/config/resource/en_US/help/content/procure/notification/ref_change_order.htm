Tips   Notification 58: Change Order

A change order is a correction or revision to an existing order, initiated when a user chooses to change a requisition whose status is Ordered. Only users with the necessary permission can change requisitions.

About the change and cancel order processes

 

If change order integration with Enterprise Resource Planning (ERP) systems is not enabled in your configuration, users cannot change any line items on requisitions or purchase orders that would result in a change order being pushed to the ERPs.

When change order integration is disabled after a change order has been created but before it has been sent to an ERP, this notification is emailed to users who have the PurchaseOrderChangeEmail permission. The authorized user must cancel the change order.
