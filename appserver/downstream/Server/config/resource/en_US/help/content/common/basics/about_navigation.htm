Tips   About navigating in My Supply Cabinet

You can access the Home page from any screen by clicking Home in the Command bar.

Links in the Navigation Panel flash at a set interval to alert you when you have pending tasks. For example, if you receive requests that require your approval, the Approve Requests link flashes for several seconds, and will continue to flash at intervals.

If you expand or collapse the Navigation Panel or sections in it, the settings apply until you click the arrows again.

It's recommended you navigate in My Supply Cabinet using the Navigation Panel, Command bar, or My Supply Cabinet's other navigational methods.

My Supply Cabinet provides standard ways to navigate through tasks and screens:

Navigation Panel. This area contains links you can click to complete a task; create and work with requests and documents, and monitor their status; explore catalogs and folders; and run searches and reports. The links displayed can vary, depending on configuration and assigned roles and permissions.

  • On many screens, the Navigation Panel groups related links in sections; for example, links to create requests and documents are grouped under a Create section. On other screens, the Navigation Panel displays links for each step in the current task; for example, when creating a requisition, the links correspond to the three steps of that task.
  • On some screens, you can click an arrow in the border of the Navigation Panel to expand or collapse the Navigation Panel itself. Click the arrow that points left to collapse or hide the Navigation Panel; click the arrow that points right to expand or display it.
  • Click a section arrow to expand or collapse that section. For example, if you rarely run reports, click the down arrow next to Report to collapse that section and hide the link to run reports; the arrow changes to indicate the section is collapsed. Click the arrow again to expand the section and display the link to run reports.

Command bar. This area contains a link that displays information about your current session, as well as buttons to navigate to the Home page and the online help, and to log out. It also provides buttons to:

  • View and change preferences and other personal profile information
  • Hide or display the Inline Tips, and their direct access to online help
  • Display amounts in your default currency or a foreign currency

Other methods. Many screens contain other methods for navigating:

  • Some screens contain a line of links, often called a breadcrumb, that you can click to move to the next or previous step in a task.
  • Many line item errors are underlined to indicate they are linked to the screen on which you can correct the error. For example, you could click The line item contains one or more invalid fields to display the screen where you must correct the invalid fields.
