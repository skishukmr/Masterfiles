Global Catalog reports

These reports list the commodity codes and suppliers of catalog items. Use these reports to manage your My Supply Cabinet product catalog. The catalog data used in these reports is updated by the GenerateReportingEntries scheduled task. For more information about this task, see the My Supply Cabinet Configuration Reference Guide.

This report Displays
Catalog by Commodity Code A list of active catalog items in the My Supply Cabinet product catalog, sorted by commodity code. You can choose which commodities and supplier to include or include all commodities and suppliers.
Catalog by Supplier A list of active catalog items in the My Supply Cabinet product catalog, sorted by supplier. You can choose which commodities and supplier to include or include all commodities and suppliers.
