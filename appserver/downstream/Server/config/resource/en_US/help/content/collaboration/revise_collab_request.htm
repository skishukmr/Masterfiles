Tips   Edit a collaboration request

When viewing proposals on the Collaboration tab, or when viewing the details of a collaboration request, you can revise the collaboration request and resend it to the supplier, or you can create an alternative request that is sent in addition to the original. Revised and alternative requests do not go through the pre-collaboration approval flow again.

When you create an alternative request, the original request is still active. If you want to replace the original request with the alternative request, you should cancel the original request.

When you revise a request, you can no longer accept any proposals that were sent for the original request. Any proposals the suppliers are currently composing in response to the original request become obsolete.

Screen details

The Collaborating tab of a requisition displays a list of the suppliers to whom the collaboration request was sent and the status of their proposals. You can display the details of the collaboration request by clicking the status of one of the proposals listed on this tab.

 
  1. Take an action:
    • If you are viewing the proposals on the Collaboration tab, click the name of the supplier to whom you want to send a revised request, and then choose Revise Request or Alternative Request from the pull-down menu, depending on whether you want to revise the original request or create an additional request.
    • If you are viewing the request details, click Actions, and then choose Revise Request or Alternative Request from the pull-down menu.
  2. In the Select Items screen, select the collaborative line item(s) you want to include in the request by clicking Included next to those items. To exclude an item that is currently set to be included, click Excluded next to it. Note that when the supplier creates a proposal, this item will still appear on the Select Items screen but will be set to Excluded.
  3. To edit a line item, click its Included or Excluded button, and then choose Edit from the pull-down menu.
  4. Click Next.
  5. In the Complete And Submit screen, verify and modify your settings just as you did when you created the original requisition from which the collaboration request was generated, and then click Submit.
