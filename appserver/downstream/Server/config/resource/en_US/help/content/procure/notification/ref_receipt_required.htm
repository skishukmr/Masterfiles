Tips   Notification 37: Receipt Required

My Supply Cabinet calculates when a receipt is overdue based on parameters defined by your My Supply Cabinet system administrator.

Users with the ReceiveError permission (typically purchasing agents) can query and edit all receipts in My Supply Cabinet.

Related topics

About notification messages

Notification messages
 

My Supply Cabinet emails a notification message to receipt approvers when it determines that a receipt is overdue. The first approver in the approval flow receives the initial notification. As soon as that approver submits and approves the receipt, the next approver in the flow receives the message, and so on, until all approvers have approved the receipt.
