Tips   Print a document

The Print command is available on every Request Details screen; other commands that appear on this screen vary, depending on the request type and status.

Related topics

Configure printing

View request status

View request details
 
  1. On the home page, click Status to list documents on the Status screen.
  2. Click the document ID to display the request details.
  3. Click Print to view the document in a browser window as it will appear when printed.
  4. In the browser window, select File > Print to print the document.
