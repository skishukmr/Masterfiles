Tips   View items in a favorites group

Favorite groups are handy for organizing catalog items by projects, departments, or types (such as office supplies), and make it easy to find frequently ordered items, which saves you time creating requests in the future.

When you add an item to a favorites group, a star appears beside the item in the catalog to make it easier to see.

Screen details

To display the View Favorite Items screen, select Favorites > group name on the Explore Catalog or Add Items screen.

 
  • To view additional information about an item, click any item-related link in the Product Details area.
  • To add an item to a request, click Add to Cart. If the catalog item contains fields that must be specified before ordering, you must fill in the blank fields before My Supply Cabinet adds the item to the request.
  • Click the lock to unlock the group and browse or search the entire catalog.

    A lock displays beside the favorites group to indicate that you are viewing the contents of that group only.
