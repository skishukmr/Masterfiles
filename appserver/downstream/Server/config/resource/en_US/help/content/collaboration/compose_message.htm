Tips   Compose a message

When collaborating on a request, the collaboration team in the buying organization can exchange messages with the supplier. Messages are posted on the message board for this request or proposal, as well as being sent to collaboration team members and the supplier by email notification.

If a collaboration team member creates a message that is not specified for supplier viewing, only collaboration team members can view it.

Screen details

The Compose Message screen appears when you click the New Message button or the Send Message menu command in various screens.

 
  1. In the Subject text box, enter a subject that summarizes the content of the message.
  2. Click Yes to make this message visible to anyone who can view the request, including the supplier, or click No to make this message visible only to the collaboration team.

    3. This option is available to members of the collaboration team only, not to suppliers.

  3. In the Message box, enter your message.
  4. To add an attachment, click Add, and then specify the file to attach.
  5. When you are ready to send the message, click Send.

The message displays on the message board for this request or proposal and is sent as an email notification to members of the collaboration team.
