Tips   Confirm saving a catalog search

When you change an existing catalog search, you must confirm that you want to replace the existing search settings with the new ones.

Screen details

You confirm saving a catalog search on the Save Catalog Search screen. To display this screen while viewing catalog search results, click Options > Saved searches > Save current search, enter the name of an existing search (to replace the previous search criteria with your existing criteria), and click Save.

 
  • Click OK to overwrite the existing search with the new settings.
  • Click Cancel to return to the previous screen to enter a new name for the search. If you enter a new name, the new search appears in your list of saved searches.
