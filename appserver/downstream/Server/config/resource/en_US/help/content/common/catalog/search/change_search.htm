Tips   Change a catalog search

Once you save a catalog search, you can change it to meet your business needs. For example, if your company decides to impose spending limits for certain items, you can change the catalog search by adding price limits (to find only those items that meet your company's price guidelines).

When you change an existing catalog search, you must confirm that you want to replace the existing search settings with the new ones.

Screen details

You change a saved catalog search from the Save Catalog Search screen. To display the Save Catalog Search screen, while viewing catalog search results, click Options > Saved searches > Save current search.
 
  1. Select Options > Run <search name> to rerun the search and display the search option settings.
  2. Change your search settings.
  3. Click Search to run the search using the changed settings.
  4. Select Options > Save Current Search. The Save Catalog Search screen displays.
  5. Re-enter the name of the existing search that you are changing. Note: If you do not enter the existing search name exactly, you will create a new search instead of changing the existing one.
  6. Click Save. The Save Catalog Search screen displays, and prompts you to replace the existing settings with your changed ones.
  7. Click OK to confirm the changes and save the existing search with the new settings.
    • Click Cancel to stop making changes.
    • If desired, enter a new search name and click Save to create a new search instead of changing the existing one.
    • Click Cancel again to suspend all changes and return to the original screen.

Alternate method:

  1. Enter new search options and click Search.
  2. Select Options > Save Current Search.
  3. Enter the name of the existing search you wish to change.
  4. Click Save. The Save Catalog Search screen displays and asks you to confirm that you want to save the search using the existing name.
  5. Click OK to confirm the changes and save the existing search using the new settings.
    • Click Cancel to stop making changes.
    • If desired, enter a new search name and click Save to create a new search instead of changing the existing one.
    • Click Cancel again to suspend all changes and return to the original screen.
