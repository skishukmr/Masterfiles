Tips   Create a custom value

Your My Supply Cabinet system administrator can configure Chooser screens to include additional buttons and functionality, which can result in additional screens used to create new objects (such as the values in a chooser). For example, the Choose Value for Supplier screen can be configured to include a New button for adding new suppliers. Clicking New displays a screen for creating a new supplier. See your system administrator for more information.

Related topics

Choose one or more values

 
Screen details

To display the Create value screen, select New on a Chooser screen. The New button appears only if configured by your My Supply Cabinet system administrator.
 
  1. Provide the requested information about the new value.
    • Click any link for more information, or to select other values.
  2. Click Save to add the new value to the list on the Chooser screen, or Cancel to return to the Chooser screen without adding the value.
