Tips   Decline a collaboration request

At any time during collaboration, you can decline the collaboration request, even if you have already created a proposal in response to the request.

Screen details

To display the Declining Collaboration Request screen, click Respond when viewing the collaboration request details in your inbox, and then choose Decline Request from the pull-down menu.
 
  1. In the Reason for Decline list, select a reason for declining this request.
  2. Optionally, enter a further explanation of why you are declining the request.
  3. Click OK to return to the collaboration request details.
  4. If you previously declined a request, you can undo your action by clicking Respond and then choosing Undo Decline from the pull-down menu.
