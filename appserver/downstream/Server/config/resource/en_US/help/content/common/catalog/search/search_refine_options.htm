Tips   Add search refinement options

Searching is not case sensitive. If you enter the keyword Blue, the search returns items that contain Blue or blue.

For automatic wildcard searching, enter characters for text searches or alphanumeric characters for part number searches. By default, you must enter a minimum of three consecutive characters, but your company can set a different minimum. See your system administrator for more information. For text searches, enter a maximum of 20s0 characters.

To remove a search field from the current display, click Options to display the Options pull-down menu and deselect the option.

The ... notation indicates that there are more categories or suppliers than the ones currently listed on the screen. Drill down or use the search refinement options to view additional categories or suppliers.

When searching for catalog items to add to a request, you can add several search fields to create more powerful searches.

  1. Click Options to display the Options pull-down menu.
  2. Select Show All to add the following advanced search options to the search area, or select options individually.
  3. Enter values in the desired optional fields to refine the current results set.
  4. Click Search to display the new results set.

See about search refinement options for examples and tips to use these options to refine a search:

  • Keywords (And): find items whose descriptions contain all keywords entered.
  • Keywords (Or): find items whose descriptions contain any keywords entered.
  • Exact Phrase: find items whose descriptions contain the text exactly as you entered it.
  • Exclude Words: omit items whose descriptions contain any of the specified words. Use a space to separate words to exclude from the search.
  • Supplier Part #: find items whose supplier part number matches the value entered. You can also use a stem search to find supplier part numbers.
  • Mfg. Part #: find items whose manufacturing part number matches the value entered.
  • Price: find items in a specific price range. Enter the upper or lower limit, or enter both upper and lower limits to narrow the results set. The correct currency format is applied to your entry, based on your default currency.
  • ETA (days):find items whose estimated time of arrival falls within the number of days entered.
  • Favorites: select an item you have previously added to a favorites group.
  • Contract: select an item from a contract that you have permission to access.
  • Supplier: find items from the selected supplier or an Unspecified supplier.
  • Manufacturer: find items from the selected manufacturer.
  • Type Name: find items with parametric data of a particular type.
    • Select the name from the Type Name pull-down menu to display the additional search fields.
    • Enter the additional parametric type attribute values before you search.
  • When you display parametric data fields, the Type Name field is locked and highlights to indicate that your search is confined to items of that type. Click the lock to unlock the search and reset the catalog.
  • Language: find items listed in the selected language (for multi-language catalogs).
