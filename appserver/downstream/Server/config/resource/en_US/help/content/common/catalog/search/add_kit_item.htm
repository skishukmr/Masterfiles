Tips   Add catalog items to a kit

The following kinds of catalog items cannot be added to catalog kits: existing kits, non-catalog items, non-catalog items created by no-release contracts, and PunchOut items.

Some of the fields, buttons, and links described in the linked help topics are not available when you are using the search screens to create and edit catalog kits. For example, you cannot add an item to a Favorites group while creating a kit.

Screen details

To display the Select Catalog Item screen when creating or editing catalog kits, click [select] on the Catalog Kits - Edit or Catalog Kits - Create screen.
 
  1. Search for catalog items to add to the catalog kit you are creating or editing:
    • Browse a catalog category.
    • Add search refinement options.
    • Use a simple text search for catalog items.
  2. View search results.

    If you choose a specific supplier for the kit, only catalog items from that supplier appear. If you want the kit to contain catalog items from more than one supplier, choose Unspecified supplier for the kit supplier.

  3. To view catalog items details, click the item's link.
    • To view additional information about an item, click any item-related link in the Product Details area. If the supplier is Unspecified, the supplier details screen will not contain any information.
    • To return to the previous screen, click back.
  4. Click Select to return to the previous screen and add the item to a catalog kit, or click Exit to return to the catalog kits screens without choosing an item.
