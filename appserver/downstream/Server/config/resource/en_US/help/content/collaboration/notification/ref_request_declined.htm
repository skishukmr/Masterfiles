Tips   Notification 95: Collaboration Request Declined

Members of the collaboration team receive email notifications related to the collaboration process and can take actions on the collaboration documents. For more information, see the Ariba Category Procurement Guide.

About collaboration
 

When a supplier declines a collaboration request, My Supply Cabinet sends this notification message to members of the collaboration team.

This notification message is for information only. It does not require any action.
