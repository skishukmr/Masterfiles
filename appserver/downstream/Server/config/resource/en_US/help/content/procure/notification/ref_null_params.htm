Tips   Notification 23: Null or Missing Parameters

Resolving this problem typically requires manual intervention. For example, the purchasing agent (or other administrator) can change the original request in My Supply Cabinet and try again. Or the purchasing agent might try to force the order. For more information about resolving such errors, see the My Supply Cabinet Procurement Guide.

About notification messages

Notification messages
 

When an order transmission fails because information is missing from the order, My Supply Cabinet emails a notification message to users who have the OrderFailureImmediateEmail permission (typically purchasing agents). The message indicates the order number, the user who created the order, and the null or missing parameter, such as contact information.
