Tips   Change the number of search results that display

The Show Results field is hidden until you browse or search for an item. After browsing or searching, you see the Show Results (per page) field, and an additional View By field.

When you change the Show Results settings, the page refreshes using the new settings.

When searching for catalog items, you can change the number of search results that display per page.

  1. Search for an item to display the Show Results pull-down menu in the search controls region.
  2. Click the Show Results pull-down arrow and select a different number of results to display.
