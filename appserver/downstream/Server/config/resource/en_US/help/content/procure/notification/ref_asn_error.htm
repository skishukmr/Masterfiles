Tips   Notification 75: Advanced Ship Notice Error
This is an informational message. It does not require any action.

About notification messages

Notification messages
 

My Supply Cabinet notifies users with AdvancedShipNoticeErrorNotification permission when a ship notice message is in a different partition than the purchase order. My Supply Cabinet does not save or update the incorrect purchase order.
