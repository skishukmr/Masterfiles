Tips   Save a catalog search

When you save a search, you save the query, rather than a fixed group of items. (The same search may yield different results over time, as items in the catalog change.)

A saved search does not encompass values specified for the Favorites or Contracts pull-down menus, because the items in these lists can vary. Even if your search includes these fields (that is, the fields are locked when you save the search), they are not included in the query the next time you run the saved search.

Screen details

To display the Save Catalog Search screen, enter your search criteria and run a search, then click Options > Saved searches > Save current search.

 
  1. On either the Explore Catalog or Add Items screen, enter keywords or other refinement options and click Search.
  2. When you are satisfied with your search settings, click Options to display the pull-down menu.
  3. Under Saved searches, select Save current search....
  4. The Save Catalog Search screen displays, containing a default search name that is based on the search criteria.
  5. Enter a unique name for the search. For example, “QA office supplies.” The search is listed under Saved Searches in the options pull-down menu.
  6. Click Save to save the search and return to the previous screen, or click Cancel to return to the previous screen without saving the search.
