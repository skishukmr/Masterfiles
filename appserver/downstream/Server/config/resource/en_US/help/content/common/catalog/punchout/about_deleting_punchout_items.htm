Tips   About deleting Internet Catalog items

When you purchase items from a supplier's Internet Catalog, deleting items can affect the final cost of your order. For example, if you have an agreement with the supplier to buy one item and get a second item at a 50% discount, if you delete an item, it can change the way your order is calculated.

The request must have the status of Composing before you can delete Internet catalog items. If the request has been submitted for approval, withdraw it to return it to Composing and then delete items.

You can delete Internet Catalog items from a request using one of these methods:

  • Delete all Internet catalog items from the same supplier from the Summary screen or View Line Items screen using the Delete command.
  • Delete selected Internet catalog items from the same supplier, using the Delete Line Item screen.
  • Edit the request and delete selected line items. See the related topic: Edit Internet Catalog items.
