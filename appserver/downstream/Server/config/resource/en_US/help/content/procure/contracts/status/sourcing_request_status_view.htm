Tips   View sourcing request status

Approved and active sourcing requests are also listed on the Approved status screen.

Screen details

To display the Sourcing Request Status screen, click Sourcing in the Navigation Panel.
 

The list displays the status of spot purchase sourcing requests (SRs) and contract sourcing requests (CSRs) that have a corresponding sourcing event, or RFX, active in Ariba Enterprise Sourcing.

  • To see the complete details of a request, click an ID.
  • To view the request’s corresponding RFX in Ariba Enterprise Sourcing, click a status, such as Open.
  • To copy a CSR, click its check box and then click Copy. The new request has a status of Composing and is listed on the Composing status screen; it does not appear in this list. Modify the new request as necessary before submitting it for approval. You cannot copy an CSR whose status is Sourced. You also cannot copy SRs.
  • To move a request from the Sourcing folder to a different folder, click its check box, select a folder using the pull-down menu or chooser, and then click Move to Folder. If the request was created in a different partition than the one in which you’re working, and if the selected folder exists on that partition, the request is archived there. If the selected folder does not exist on that partition, the request is moved to the Archive Items folder on that partition.
