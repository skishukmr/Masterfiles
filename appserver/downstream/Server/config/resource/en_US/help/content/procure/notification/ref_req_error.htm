Tips   Notification 22: Requisition Error

In some cases, the status of the request reverts to Composing. The preparer (or an administrator) must correct the error and resubmit the requisition.

Related topics

About notification messages

Notification messages
 

When My Supply Cabinet is unable to push an order to an underlying ERP system, it emails a notification message to the preparer and to users who have the ERPPushFailNotificationRole permission (typically purchasing agents).

The message identifies the order that caused the error, the ERP system involved, the specific errors that occurred, and the action required to resolve the error.
