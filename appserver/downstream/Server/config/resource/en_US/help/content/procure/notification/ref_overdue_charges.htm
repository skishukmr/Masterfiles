Tips   Notification 45: Overdue PCard Charges

If for some reason a PCard appears to have overdue charges and no users have authority to use it, My Supply Cabinet emails the message to the purchasing card manager.

Related topics

About notification messages

Notification messages
 

If you have overdue charges on a corporate purchasing card (PCard) that have not been explained, My Supply Cabinet emails a notification message requesting that you explain the charges. The message includes information about the unexplained charges, including merchant name, date, and amount.

You must log in to explain the charges. If you do not, My Supply Cabinet escalates the request to the purchasing card manager.
